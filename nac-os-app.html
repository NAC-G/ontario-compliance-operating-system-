⬚ QR Code Manager
How QR codes work
Each QR code encodes a link to this app pre-loaded to that machine's daily log entry form. Print, laminate, and place in the cab. Operator scans → app opens → machine is pre-selected → they fill in end hours and checks. Works from any phone browser, no app install needed.
⬚ QR Code Manager
Scan to open app → machine pre-selected → straight to log entry
ℹ How QR Codes Work
Step 1 — Print
Click "Print Label" on any machine card. Print the label and laminate it. Stick it in the cab or on the machine frame.
Step 2 — Scan
Operator scans the QR code with their phone camera. The app opens directly — machine is pre-selected, date is set, ready for log entry.
Step 3 — Log
Operator fills in the pre-use checks, end hours, and fuel. Saves in seconds. No paper. No manual machine selection. Data stays on the device.
⬚ QR Code Generator
Each QR code opens the NAC OS Fleet Logbook directly to the New Log Entry form with that machine pre-selected. Print, laminate, and mount in the cab. Operator scans → log opens → machine is already filled in.
⚙ Base URL Configuration
Set the URL of where your NAC OS app is deployed. This is the URL encoded into every QR code.
QR codes will encode: [base-url]?machine=[MACHINE-ID]
Example: https://naturalalternatives.ca/nac-os-app.html?machine=BH-580SN-01
